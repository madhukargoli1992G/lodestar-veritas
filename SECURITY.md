# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please do not create a public issue.

Instead, contact the project maintainer through GitHub with the following information:

* Description of the vulnerability
* Steps to reproduce
* Potential impact
* Suggested mitigation (if available)

We will investigate and respond as quickly as possible.

## Supported Versions

| Version        | Supported |
| -------------- | --------- |
| 1.x            | ✅         |
| Older Versions | ❌         |

Thank you for helping keep Lodestar Veritas secure.
